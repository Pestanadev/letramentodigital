Bem-vindo ao Letramento Digital

Este projeto de extensão pretende capacitar os idosos com habilidades digitais essenciais para promover uma participação mais ativa nesta era digital, fortalecendo a conexão entre gerações. O projeto é desenvolvido pela prof Iara Carnevale de Almeida - docente no curso de Sistemas para Internet, UniALFA Umuarama / PR - apoiado pelos alunos deste curso.
